# iFruite Bot — Instagram AI Sales Assistant

AI-бот для магазину Apple техніки. Моніторить Instagram Direct через AdsPower, відповідає на питання про наявність та ціни з Airtable, надсилає на затвердження менеджеру в Telegram.

## Архітектура

```
┌─────────────┐     ┌──────────────┐     ┌───────────┐
│  AdsPower    │────▶│  Instagram   │────▶│  Airtable  │
│  (браузер)   │     │  Extractor   │     │  (каталог) │
└─────────────┘     └──────────────┘     └─────┬─────┘
                                               │
                                         ┌─────▼─────┐
                                         │  OpenAI    │
                                         │  (відповідь)│
                                         └─────┬─────┘
                                               │
                    ┌──────────────┐     ┌─────▼─────┐
                    │  AdsPower    │◀────│  Telegram  │
                    │  (відправка) │     │  Bot       │
                    └──────────────┘     │  OK/EDIT/  │
                                        │  SKIP      │
                                        └────────────┘
```

## Потік роботи

1. Кожні 5 хв сканує Instagram Direct через AdsPower
2. Знаходить непрочитані повідомлення
3. Шукає релевантні товари в Airtable
4. OpenAI генерує відповідь українською з цінами та наявністю
5. Відправляє чернетку в Telegram чат з кнопками OK / EDIT / SKIP
6. Менеджер затверджує або коригує
7. Відповідь відправляється клієнту в Instagram

## Налаштування

```bash
cp .env.example .env
# Заповнити змінні в .env
npm install

```
